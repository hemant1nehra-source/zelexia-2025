# ZELEXIA 2025 — Thapar University

Official event website for **Zelexia**, the biggest DJ Night at Thapar University, held at the Main Auditorium, Patiala.

## 🌐 Live Website
[Click here to view the live site](https://hemant1nehra.github.io/zelexia-2025)

## 📌 About
Zelexia is a neon-themed DJ event organized by students of Thapar University.
This website was built to provide event information and handle student registrations.

## ✨ Features
- Animated neon particle effects
- DJ lineup & performance schedule
- Student registration form
- Sponsor section (White Bull Energy Drink)
- Fully responsive (mobile + desktop)

## 🛠 Built With
- HTML5
- CSS3 (animations, gradients, neon effects)
- Vanilla JavaScript (Canvas particles, form handling)
- No frameworks — pure code
